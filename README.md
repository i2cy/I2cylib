# 主要包含
 - `ICCode` i2cy常用的混淆算法
 - `Dynkey` 动态验证密匙生成/验证工具
 - `SQLiteDB` SQLite3数据库面向对象式API
 - `ICFat64` 类FAT虚拟文件系统
 - `I2TCP` 高度封装的用户层通讯协议
 - `PID` 异步PID模组
 - `utils` 各种常用的小工具

# 安装方法
`pip install i2cylib`

# 内嵌命令行工具
 - `icen` 基于ICCode混淆算法的文件加密工具
 - `i2cydbserver` 基于SQLite的数据库服务端
 - `i2scan` 端口扫描、系统推断工具

# 环境需求
`Python3.6+`
